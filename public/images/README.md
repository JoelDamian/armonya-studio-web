# Images

Place the design images here with these names so the site displays correctly:

- `hero.png` — Hero section (Harmony In Design / interior)
- `about.png` — About section (vase and pampas)
- `portfolio.png` — Portfolio grid (use same or separate images per project)
- `contact.png` — Optional contact section image
- `disciplines.png` — Creative disciplines card image
- `cta.png` — Optional CTA section background
- `footer.png` — Optional footer image

You can copy the provided design assets from your Cursor assets folder into this directory and rename them as above.
